# 举例：画自己的页面

装好之后，怎么画一个**自己的**页面？这篇用最简单的例子手把手教。

---

## 先跑通最小例子

仓库自带一个比反馈页简单得多的入门例子 `hello-page.mjs`：

```bash
cd ~/Desktop/axure-mac-bridge/svg-generator
node examples/hello-page.mjs
```

输出 `~/Desktop/原型/hello-page.svg`，是一个"个人中心"页面：头像、名字、三个设置项、退出按钮。

打开看看：

```bash
open -a Safari ~/Desktop/原型/hello-page.svg
```

## 核心：4 个 API 就够用

所有页面都是这几样东西拼出来的。

### 1. `txt(x, top, 文字, 选项)` — 写字

```js
import { txt, colors as C } from '../lib/primitives.mjs';

// 左对齐标题
txt(24, 130, '你好', { size: 22, weight: 700, fill: C.navy })

// 居中文字（传 anchor: 'middle'，x 就是中心点）
txt(187, 744, '提交', { size: 15, fill: C.white, anchor: 'middle' })

// 右对齐
txt(359, 50, '更多', { size: 13, fill: C.primary, anchor: 'end' })
```

> ⚠️ **永远用 `txt()`，不要直接写 `<text>`**。直接写的话 Axure 会把文字全堆到顶部。原因见 [04-Axure-SVG-怪癖.md](04-Axure-SVG-怪癖.md)。

参数说明：
- `x`：横坐标（左对齐时=左边，居中时=中心，右对齐时=右边）
- `top`：文字**顶边**的 y（不是基线，lib 帮你换算）
- 选项：`size` 字号、`weight` 粗细（400/500/700）、`fill` 颜色、`anchor` 对齐

### 2. `rect(x, y, 宽, 高, 颜色, 选项)` — 画矩形

```js
import { rect, colors as C } from '../lib/primitives.mjs';

// 输入框（灰底 + 边框 + 圆角）
rect(16, 228, 343, 140, C.inputBg, { stroke: C.border, rx: 8 })

// 实色按钮
rect(16, 728, 343, 48, C.primary, { rx: 8 })

// 分割线（1px 高的矩形）
rect(16, 156, 343, 1, C.border)
```

### 3. 手机外壳 — `phoneFrame / statusBar / navBar / homeIndicator`

```js
import { phoneFrame, statusBar, navBar, homeIndicator, PHONE_W, PHONE_H } from '../lib/phone.mjs';

const frame = phoneFrame(60, 60, 'Page 1');  // 在 (60,60) 放一个手机框
push(frame.begin);
push(statusBar());                            // 顶部 9:41 状态栏
push(navBar('页面标题', '右上按钮'));          // 导航栏
//  ... 你的内容 ...
push(homeIndicator());                        // 底部小条
push(frame.end);
```

### 4. `specPanel(...)` — 右侧需求说明（可选）

如果要"左图右说明"排版：

```js
import { specPanel } from '../lib/spec-panel.mjs';

push(specPanel(specX, rowY, 1, '页面标题', [
  { heading: '功能目标', items: ['一句话目标'] },
  { heading: '页面元素', items: [
    '• 顶栏：返回 + 标题',     // • 开头 = 一级列表
    '• 主体：表单',
  ]},
  { heading: '字段', items: [
    '- name：必填',            // - 开头 = 二级缩进
  ]},
]));
```

## 配色表

`colors` 对象（`import { colors as C }`）：

| 名字 | 色值 | 用途 |
|---|---|---|
| `C.navy` | #111827 | 主文字 |
| `C.gray` | #4B5563 | 次要文字 |
| `C.light` | #9CA3AF | 占位/辅助 |
| `C.primary` | #3B82F6 | 主色（按钮、强调） |
| `C.primaryBg` | #EFF6FF | 主色浅底 |
| `C.border` | #E5E7EB | 边框 |
| `C.inputBg` | #F9FAFB | 输入框底色 |
| `C.card` | #FFFFFF | 卡片白 |
| `C.canvas` | #F5F5F7 | 画布灰底 |
| `C.red / green / amber` | - | 状态色 |

## 完整骨架（复制即用）

```js
import { writeFileSync, mkdirSync } from 'node:fs';
import { homedir } from 'node:os';
import { txt, rect, colors as C, svgHeader, svgFooter } from '../lib/primitives.mjs';
import { PHONE_W, PHONE_H, phoneFrame, statusBar, navBar, homeIndicator } from '../lib/phone.mjs';

const PAD = 60;
const out = [];
const push = (s) => out.push(s);

push(svgHeader(PHONE_W + PAD * 2, PHONE_H + PAD * 2));
push(`<rect width="100%" height="100%" fill="${C.canvas}"/>`);

const frame = phoneFrame(PAD, PAD, '我的页面');
push(frame.begin);
push(statusBar());
push(navBar('标题'));

// ====== 在这里加你的内容 ======
push(`  ${txt(24, 130, '随便写点啥', { size: 20, weight: 700, fill: C.navy })}`);
push(`  ${rect(16, 180, 343, 48, C.primary, { rx: 8 })}`);
push(`  ${txt(PHONE_W / 2, 196, '按钮', { size: 15, weight: 700, fill: C.white, anchor: 'middle' })}`);
// ==============================

push(homeIndicator());
push(frame.end);
push(svgFooter);

const outPath = `${homedir()}/Desktop/原型/my-page.svg`;
mkdirSync(`${homedir()}/Desktop/原型`, { recursive: true });
writeFileSync(outPath, out.join('\n'), 'utf8');
console.log('✅', outPath);
```

把它存成 `svg-generator/examples/my-page.mjs`，改中间那段，跑：

```bash
node examples/my-page.mjs
open -a "$HOME/Library/Application Support/axure-bridge/AxureBridge.app" --args "$HOME/Desktop/原型/my-page.svg"
# 拖文件进 Axure
```

## 坐标速记（移动端 375×812）

- 屏幕宽 `PHONE_W = 375`，高 `PHONE_H = 812`
- 状态栏：0~44
- 导航栏：44~96
- 内容区：96~798
- Home indicator：798~803
- 常用左右边距：16（内容贴边留 16px）
- 内容区可用宽度：375 - 16×2 = **343**

所以你会看到例子里大量 `rect(16, y, 343, ...)` —— 343 就是这么来的。
